# New User Panel Frontend

React UI for the Matrimonial platform User Panel.

## Prerequisites

- Node.js

## Configure API Base

Create `New User Panel Frontend/.env.local`:

```
VITE_API_URL=http://localhost:8000/api
```

## Install and Run

```powershell
cd "C:\laragon\www\marriagebureau\New User Panel Frontend"
npm install
npm run dev
```

## Build for Production

```powershell
cd "C:\laragon\www\marriagebureau\New User Panel Frontend"
npm run build
```
